## Data the app handles

Vault Calculator lets users select files from their device and import them into an encrypted local vault. The selected files are encrypted and stored in the app's private storage on the user's device.

The current version does not create accounts, upload files, sell user data, use analytics, or share vault files with third parties.

## Permissions and file access

The app uses Android's system file picker. Users choose which files the app can access. The app does not request broad all-files access.

After import, the app tries to remove the original selected file when Android allows it. If Android or the file provider does not allow deletion, the user must delete the original manually.

## Encryption and retention

Vault files are encrypted locally using a key stored in the Android Keystore. Files remain on the device until the user deletes them from the vault or uninstalls the app. Uninstalling the app deletes the vault and encryption key, which means vault files cannot be recovered.

## Data sharing

The current version does not share personal or sensitive user data.
